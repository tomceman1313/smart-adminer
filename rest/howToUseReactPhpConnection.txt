React používá php scripty přes funkci fetch();

fetch("http://localhost:4300/rest/?action=db&type=products")
		.then(response=>{
			response.text().then(_data =>{
				var data = JSON.parse(_data);
				setList(data.list);
				console.log(data.list);
			})
		})

V rest.php jsou definované třídy obsahující funkce.
pomocí _GET('neco') dostaneme action a type.
action určuje jakou třídu použijeme a type je název funkce, které se bude volat.

Takto se to dá použít pro načtení dat z databáze, nicméně pro zpracování formuláře kvůli emailu nebo insertu musíme použít knihovnu react-hook-form.

Užití react-hook-form:
	1) každý input se musí registrovat: <input placeholder="Jméno" {...register("name")} />

	2) následně se v php POSTnutá data zpřístupní: $data = json_decode(file_get_contents("php://input"), true);

	3) potom se už použití provádí normálně, jak při POST: $name = $data["name"];
